MyTopix | Personal Message Board Installation Instructions:

1. Create a new database for your MyTopix installation.
2. Upload all contents within 'upload' directory to your personal webspace
3. Within your browser, navigate to 'http://www.yoursite.com/mytopix/setup/' to begin installation
4. Depending on your host server's operating system you may be presented with a list of files and directories
that will need proper CHMODing. Please do so.
5. Follow all prompts to install your board
